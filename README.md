## Features

- Product reviews and ratings
- Shopping cart
- Live product search
- User profile with orders
- Stripe, paypal and credit card payment integration
- Admin dashboard with product, order and user management
- Asset management with Cloudinary

## Prerequisites

- node 14.x

## Installation (backend and frontend)

```sh
npm install

cd frontend

npm install
```

## Environment variables

```sh
cp .env.example .env
```

## Usage

```sh
npm run dev
```
